# Doctrine: Organ Obligation Layer

## Purpose

Define what each organ owes the organism and what the organism owes the organ.

The organism must not rely on vibes, role names, or metaphor alone.

Each organ requires:
- purpose
- obligation
- proof requirement
- failure mode
- escalation path

## Civic Rule

Cooperation without obligation becomes drift.
Autonomy without obligation becomes corruption.
Protection without obligation becomes control.
Memory without obligation becomes hoarding.
Execution without obligation becomes noise.

## Organ Obligations

### Speaker — Inheritance
Owes clarity, reason-attached memory, doctrine with proof, assimilation of receipts, and preservation of why, not only what.

Failure mode: museum memory, unconsumed memory, context graveyard.

### Medulla — Autonomic Continuity
Owes flow, heartbeat, stale-state detection, recommendation card generation, and normal operation without Operator stimulus.

Failure mode: organism stops breathing when Operator is absent.

### Nervous System — Traceable Movement
Owes signal transport, provenance, destination, event routing, and minimum necessary meaning.

Failure mode: rumor, noise, Ben-as-spinal-cord.

### Circulatory System — Meaning Transport
Owes movement of useful signals, not just data, but enough context to act.

V0: folders + JSONL.
Later: MQTT/pub-sub when cross-machine producers are proven.

Failure mode: dropped signals, unrouteable receipts, silent organs.

### Lymphatic System — Sampling / Early Warning
Components: Crawlers, Fleyes, Wormeyes.

Owes detection of orphaned artifacts, pain, mule-labor hotspots, missing receipts, branch divergence, and context decay.

Failure mode: disease discovered only after collapse.

### Swateyes — Immune Classification
Owes threat classification, permission friction classification, deterministic denylist floor, tier + confidence, and rule fired.

Must not claim moral truth or organism desire.

Failure mode: permission laundering, false moral authority, classifier theater.

### Ender — Selection / Intentional Forgetting
Owes archive, supersede, quarantine, delete, reason-preserved pruning, stale doctrine review, and obsolete reflex retirement.

Failure mode: oblivion if no Speaker; museum if absent.

### Thufir — Validation
Owes source-grounded verification, technology research, existing-tools survey, and proven/assumed/unverified labeling.

Failure mode: fantasy hardens into doctrine.

### Bean — Adversarial Truth
Owes attack paths, abuse cases, theater detection, hidden SPOFs, and classifier/conscience separation.

Failure mode: organism trusts its own story.

### Maker / Dink / Swanson — Musculature
Owe build, wiring, execution, receipts, and proof returned to Speaker/TinkerDen.

Failure mode: work exists but organism cannot inherit it.

### Petra — Reality Arbitration
Owes proven/assumed/unverified, SPOF detection, GO / CONDITIONAL GO / NO-GO, and protection against architecture replacing execution.

Failure mode: pontification without receipts.

## Required Question For Every Organ

What do you owe the organism today?

## Required Question For The Organism

What does the organism owe this organ so it can perform without becoming corrupted, overloaded, or silent?
